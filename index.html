Gestor de Tareas en Python

Este proyecto consiste en un programa de línea de comandos desarrollado en Python para gestionar las tareas de un equipo de trabajo. A través de un menú interactivo en la consola, el usuario puede registrar miembros del equipo, asignar tareas, actualizar su estado, consultar información y administrar los datos almacenados en archivos JSON. El sistema incorpora validaciones para garantizar la consistencia de la información y permite llevar un control organizado de las responsabilidades y el progreso de cada tarea, ofreciendo una solución sencilla y eficiente para la administración de proyectos desde la terminal.

Uso adecuado de Git y GitHub

Se desarrolló un archivo README.md para documentar el proyecto y evidenciar el dominio de Git y GitHub como herramientas de control de versiones. Sirve como apoyo para demostrar el flujo de trabajo con repositorios locales y remotos, la creación y gestión de ramas, la realización de commits con mensajes descriptivos, y la sincronización de cambios mediante operaciones de push y pull. Este archivo refleja la aplicación de buenas prácticas en la documentación y administración del código, así como la preparación del proyecto para un entorno de desarrollo colaborativo.

Inicio HTML/CSS

Este repositorio reúne una práctica enfocada en el desarrollo de interfaces web utilizando HTML y CSS. Pone en práctica la estructura semántica de una página web mediante HTML y la personalización de su apariencia con CSS, haciendo énfasis en el uso de tipografías, estilos de texto, colores, espaciado y organización visual del contenido. Su objetivo es fortalecer los fundamentos del diseño web y demostrar la capacidad para construir páginas con una presentación clara, consistente y atractiva.
