# 💼 Portafolio Web Personal

## 📖 Descripción

Este proyecto corresponde a mi portafolio web personal, desarrollado como parte de mi proceso de formación como **Desarrolladora Full Stack**. Su propósito es presentar información sobre mi perfil profesional, las tecnologías que estoy aprendiendo y algunos de los proyectos que he desarrollado durante mi entrenamiento.

El sitio fue construido utilizando únicamente **HTML5** y **CSS3**, aplicando buenas prácticas de estructura semántica, organización del código mediante la metodología **BEM** y un diseño limpio.

---

## 🚀 Tecnologías utilizadas

* HTML5
* CSS3
* Metodología BEM
* Git
* GitHub

---

## ✨ Características

* Estructura semántica con HTML5.
* Organización de estilos utilizando metodología BEM.
* Sección de presentación personal.
* Sección de habilidades técnicas.
* Galería de proyectos desarrollados.
* Información de contacto y redes profesionales.

## 💻 Cómo visualizar el proyecto

1. Clona este repositorio:

git clone https://github.com/Puntiana/Portafolio.git

2. Ingresa a la carpeta del proyecto:

cd Portafolio

3. Abre el archivo `index.html` en tu navegador favorito.
